# faker_rust::games::zelda

Available since version 1.7.3.

```rust
// Random Zelda game
zelda::game() //=> "Ocarina of Time"

// Random Zelda character
zelda::character() //=> "Guru-Guru"

// Random Zelda location
zelda::location() //=> "Tarrey Town"

// Random Zelda item
zelda::item() //=> "Master Sword"
```
