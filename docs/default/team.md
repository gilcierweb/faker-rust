
# faker::Team

```rust
# Random Team Creature
Team::creature() //=> "gooses"

# Random Team Name created from random US State (Address.state) prepended to a random Team Creature
Team::name() //=> "Oregon vixens"

# Random Team State
Team::state() //=> "Oregon"

# Random Team Sport
Team::sport() //=> "lacrosse"

# Random Team Mascot
Team::mascot() //=> "Hugo"
```
