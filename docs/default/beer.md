
# faker::Beer

Available since version 1.6.2.

```rust
Beer::brand() //=> "Heineken"

Beer::name() //=> "Hercules Double IPA"

Beer::style() //=> "Belgian Strong Ale"

Beer::hop() //=> "Equinox"

Beer::yeast() //=> "2278 - Czech Pils"

Beer::malts() //=> "Rye malt"

Beer::ibu() //=> "40 IBU"

Beer::alcohol() //=> "6.3%"

Beer::blg() //=> "18.5°Blg"
```
