# faker_rust::beer

available since version 1.6.2.

```rust
beer::brand() //=> "heineken"

beer::name() //=> "hercules double ipa"

beer::style() //=> "belgian strong ale"

beer::hop() //=> "equinox"

beer::yeast() //=> "2278 - czech pils"

beer::malts() //=> "rye malt"

beer::ibu() //=> "40 ibu"

beer::alcohol() //=> "6.3%"

beer::blg() //=> "18.5°blg"
```
