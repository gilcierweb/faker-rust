# faker_rust::music::music

// Music

Available since version 1.6.4.

```rust
music::key() //=> "C"

music::chord() //=> "Amaj7"

music::instrument() //=> "Ukelele"

music::band() //=> "The Beatles"

music::album() //=> "Sgt. Pepper's Lonely Hearts Club"

music::genre() //=> "Rock"

music::mambo_no_5 //=> "Monica"
```
